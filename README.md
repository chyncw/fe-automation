# Fe-automation
Step to run in you locals :
1. Make sure you're using chrome driver for Chrome 114. You can get the driver from here : https://chromedriver.storage.googleapis.com/index.html?path=114.0.5735.90/
2. Make sure you're browser version is Chrome 114.
3. Make sure you've already install Ruby.
4. Clone the repository to your locals
5. Run "bundle install" in your terminal.
6. Run "gem install bundler" in your terminal.
8. Run all the scenario by using syntax "cucumber".
